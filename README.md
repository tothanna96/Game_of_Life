# Game_of_Life

# Használt technológiák:

- Backend: SpringBoot - Java
- Frontend: Angular (futtatása cmd-ben: ng serve --proxy-config proxy.conf.json)

# Megvalósítotott részfeladatok: 

- Frontend és Backend különválasztása.
- Be lehet tölteni .lif formátumú fájlokat, a #-el kezdődő sorokat kihagyva.
- Az egyes generációkból kiszámolja a következőt a szabályok alapján.
- A generációkat megjeleníti grafikusan.
- Van egy léptető gomb (Step), egy automata lejátszó gomb (Play)
  és egy automata lejátszást megállító gomb (Stop).
- Kijelzi, hogy hányadik generációnál járunk a kiinduló állapothoz képest.
